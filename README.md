## Compilation:
```
$ javac akhibara.cli/*.java
& javac akhibara.cli/*.java
```
*(if you have difficulties to compile, you can use the officials executables files)* 

## Libraries:
OpenCSV 
Swing

## Maven:

pom.xml :
```xml
<dependencies>
  	<dependency>
  		<groupId>com.opencsv</groupId>
  		<artifactId>opencsv</artifactId>
  		<version>5.5.2</version>
  	</dependency>
 </dependencies>
```
```
$ mvn compile
```
